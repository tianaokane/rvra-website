# Roe Valley Residents Association Website

A responsive community website for **Roe Valley Residents Association (RVRA)**, a community-led organisation based in Limavady, Northern Ireland.

The website shares information about RVRA’s projects, facilities, events, team, community support work, and ways for people to get involved.

## Live Website

The site is hosted using **GitHub Pages**.

**Live site:**
`https://tianaokane.github.io/rvra-website/`

Replace the link above with the final GitHub Pages URL.

## About the Project

This website was designed to give RVRA a welcoming, accessible, and easy-to-navigate online presence.

It includes pages for:

* Home
* About Us
* Our Story
* Our Projects
* Marie’s Beacon Fund
* Our Team
* Facilities
* News & Events
* Get Involved
* Privacy Policy
* Accessibility

The design uses a bright, family-friendly visual style with RVRA’s blue, green, and orange branding.

## Features

* Responsive layout for desktop, tablet, and mobile

## Running the Website Locally

Because this is a static website, it can be opened directly in a browser.

1. Download or clone the repository.
2. Open `index.html` in your browser.

For a better development experience, use the **Live Server** extension in VS Code.

## Deployment

This site is deployed using **GitHub Pages**.

To deploy:

1. Push the project files to a GitHub repository.
2. Go to the repository settings.
3. Open **Pages**.
4. Set the source branch to `main`.
5. Save the settings.
6. GitHub will generate a live website link.

## Accessibility

The site aims to follow good accessibility practice, including:

* Semantic HTML structure
* Clear navigation
* Responsive layouts
* Alt text on meaningful images
* Keyboard-friendly links and buttons
* Readable colour contrast
* Plain-language content

## Notes for Future Updates

When updating the website, remember to:

* Keep footer links consistent across all pages
* Update the year in the footer if needed
* Replace image placeholders with real RVRA images
* Check all internal links after renaming files
* Update the Privacy Policy if forms, analytics, booking tools, or embedded services are added
* Keep accessibility in mind when adding new images, colours, forms, or scripts
